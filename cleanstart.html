Enable background blur for the best CleanStart experience.

CleanStart just got a complete redesign, which includes background blur. Firefox disables background blur by default, so here's how to enable it:

  1. Go to the page about:config.
  2. If a Proceed with Caution page appears, click the button Accept the Risk and Continue.
  3. Type layout.css.backdrop-filter.enabled into the search box.
  4. Double-click the preference to set it to true.
