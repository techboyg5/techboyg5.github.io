--- layout: default title: The Real-time HTML Editor is gone. But don't worry. ---

The Real-time HTML Editor project has merged into the Web Development Tutorial, and is now called the Code Editor.

On each of it's pages, you'll find a new "Code Editor" link which opens the new Code Editor.

How is the Code Editor better than the Real-time HTML Editor?

The Code Editor is better than the Real-time HTML Editor because it has seperate tabs for HTML, CSS and JavaScript, and is integrated right into the Web Development Tutorial.
