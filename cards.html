Website Cards by techboyg5

These cards let you easily reference one web page on another. This is what one looks like:

To add the cards to your website, add the following code:

<iframe class="g5cards" src="https://techboyg5-php.herokuapp.com/cards.php?favicon=aHR0cHM6Ly9wcm9qZWN0cy50ZWNoYm95ZzVibG9nLmNvbS9mYXZpY29uLmljbw==&href=aHR0cHM6Ly9wcm9qZWN0cy50ZWNoYm95ZzVibG9nLmNvbS8=&title=techboyg5%27s%20Projects&desc=No%20matter%20what%20you%27re%20up%20to,%20techboyg5%27s%20projects%20have%20a%20lot%20to%20offer." style="border:none;width:100%"></iframe>

This generates the example above.

Customization

You can customize the card by changing the URL parameters. They are:

  • favicon: The path to the site's favicon. Usually this is https://website.com/favicon.ico. Base64 encoded.
  • href: The href of the card. Base64 encoded.
  • title: The title of the card. Not Base64 encoded.
  • desc: The description of the card. Not Base64 encoded.
