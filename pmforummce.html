Pale Moon Forum - Migrating to TinyMCE

Dear Moonchild Productions,

Did you know that you can completely change your editor with just one file change and one setting change?? This editor is called TinyMCE. Installing TinyMCE onto a phpBB-based forum removes the requirement for learning BBCode and makes editing a whole lot easier. This editor looks like this:

Doesn't it look amazing? Here's how you can install it on your phpBB-based forum.

Installing TinyMCE into phpBB

These instructions have been tested with phpBB version 3.3.0.

Allowing template files to be changed

By default, any changes to template files (styles/prosilver/template) will not affect your forum. You have to enable this functionality in the ACP.

Go to your ACP. Authenticate. Under Server Configuration, select Load settings. Choose Yes for "Recompile stale style components", the select Submit. You can now close the ACP.

Now you can edit the template files.

Removing the default editing controls

Using FTP, open styles/prosilver/template/posting_editor.html (in your forum's directory) in a text editor.

The first step is to remove the default editing controls. You can remove the default editing controls by removing this line:

<!-- INCLUDE posting_buttons.html -->

Now, save the file. The editing buttons should disappear on all posting screens now.

Installing TinyMCE

Now, it's time to install TinyMCE. Using FTP, open styles/prosilver/template/posting_editor.html (in your forum's directory) in a text editor.

Add the following lines of code anywhere in the file:

<script src="https://cdn.tiny.cloud/1/t138ohs7zr1z7n41wswihq3hd60np2ms3ulzkti97vfipbt2/tinymce/5/tinymce.min.js" referrerpolicy="origin"></script>
<script>
    tinymce.init({
        selector: 'textarea',
        plugins: 'code bbcode'
    });
    </script>

Don't understand any of this? Here is each line in detail:

  • The external script is the TinyMCE editor itself. Note that TinyMCE requires an API key to be used which is tied to one or more domains. This API key works on the following domains: localhost, forum.palemoon.org, and techboyg5.github.io.
  • The tinymce.init code loads TinyMCE on the textarea. It also has two plugins: code, which lets you edit the raw code), and bbcode, which switches the editor from HTML to BBCode.

Now, save the file. The default editor should now be replaced with the TinyMCE editor.
