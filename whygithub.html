Why you should join GitHub

Even if you won't host repositories on GitHub, joining GitHub is a great thing to do and helps the community. Here's why.

Sign-up is free and easy

Signing up for GitHub is very easy. Just type a unique username, your email address and a password. Unlike most sign-up forms, GitHub's form dosen't ask for your real name or date of birth. And it is completely free to use GitHub.

You can see how things work

Do you use some free and open-source software whose repository is hosted on GitHub? Curious how something in it works? Just browse the software's repository.

You can help software developers very easily

There are many ways that you can help the community of software developers very easily by joing GitHub.
