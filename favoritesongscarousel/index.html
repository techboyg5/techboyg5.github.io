My Favorite Songs Carousel

An embeddable object that scrolls through my favorite songs. Can be used as a carousel example.

Embed Code

This carousel is intended for use in examples. You can put this code in your page to embed the carousel above:

<iframe src="https://techboyg5.github.io/favoritesongscarousel/carousel.html" width="640" height="360" style="border:none"></iframe>

Use a 16:9 aspect ratio. 640x360 is 16:9 and is perfect for examples.
