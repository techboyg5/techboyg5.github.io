--- layout: default title: My Favorite Songs Carousel

An embeddable media object that scrolls through my favorite songs. Intended to be used as a carousel example.

Put this code in your website or blog:

<iframe src="https://techboyg5.github.io/favoritesongscarousel/carousel.html" width="640" height="360" style="border:none" allowfullscreen></iframe>

Use a 16:9 aspect ratio. 640x360 is 16:9 and is perfect for examples.
